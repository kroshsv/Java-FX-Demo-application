Java FX Demo application with custom controls

Программное обеспечение необходимое для работы данного приложения:

 - Java 13.0.1

 - open JavaFX 11.0.2

 - Apache NetBeans IDE 11.2

 - JavaFX Scene Builder 11

 - TIBCO Jaspersoft Studio 6.11.0
